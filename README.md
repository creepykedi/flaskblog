# flaskblog

Flaskblog - simple blog project made in Flask. Uses SQLAlchemy and Bootstrap4.
You can:
- create user, 
- login, 
- write/update/delete posts,
- get all post by selected user,
- there's profile page with customizable profile picture,
- password reset.
